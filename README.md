Разработка методики цифрового документирования наследия исторических городов на примере Северо-Западного Федерального округа.
Методика функционирует по следующей логике:
1. Выбор потенциального населенного пункта и определение целей и задач исследования. На этом этапе исследовательской работы формируются цели и задачи методики, а также определяются ключевые параметры, которые будут оцениваться в ходе исследования. Исходные данные о материальном и нематериальном наследии.
2. Сбор необходимых и дополнительных данных. Необходимо собрать важные исторические и культурные данные о городе, на основе которых будет создана база данных. 
3. Следующий шаг включает в себя проверку полученных данных включает в себя проверку полученных данных на актуальность и правдивость.
4. В данном пункте подразумевается систематизация и цифровизация полученных данных в систему, которая отображает структуру классификации выявленных факторов идентичности исторических городов. Структура базы данных состоит из системы таблиц объектов наследия, где для каждого элемента присвоен идентификатор и расписаны критерии и параметры заполнения информации.
5. На данном этапе предполагается разработка специальной цифровой базы данных, которая будет использована для отображения собранной информации. Входными данными на этом этапе являются уже ранее обработанные данные и результаты их анализа.
Результаты документирования в базу данных выгружаются в формате XML и dbase. Ожидается, что алгоритм будет способствовать расширению методики и на другие исторические города Северо-Западного федерального округа, сохраняя при этом стабильность и воспроизводимость результатов.
