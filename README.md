Разработка методики цифрового документирования наследия исторических городов на примере Северо-Западного Федерального округа
В выпускной квалификационной работе представлена методика цифрового документирования наследия исторических городов на примере Северо-Западного Федерального округа. Методика направлена на создание единой системы хранения информации в цифровом виде, систематизацию данных и дальнейшее их использование для работы с объектами, представляющими ценность.
